Nuctech Transmission Dashboard

Project Summary

Nuctech Transmission Dashboard is a real-time monitoring application designed to track and visualize container scanner logs from Nuctech systems. The application features live data updates, comprehensive statistics overview, and detailed log viewing with advanced filtering capabilities. Built with a modern tech stack, it provides an intuitive interface for monitoring transmission activities and system performance.

Technologies Used

  • Frontend: React, TypeScript, Ant Design
  • Backend: Node.js, Express.js
  • Database: PostgreSQL
  • Real-Time Communication: Socket.IO
  • UI Framework: Ant Design
  • Development Tools: Git, Postman

Challenge & Solution

Challenge

The primary challenge was to create a system capable of handling and displaying real-time log data from multiple Nuctech scanners simultaneously. This required efficient data processing, real-time updates to multiple clients, and an intuitive interface for filtering and analyzing large volumes of log data.

Solution

A WebSocket-based architecture using Socket.IO was implemented to enable real-time data transmission between the server and multiple dashboard clients. The backend efficiently processes incoming log data and broadcasts updates to all connected clients. The frontend utilizes React with Ant Design components to create a responsive and user-friendly interface with advanced filtering and search capabilities.

Main Features

  • Real-Time Log Monitoring: Live updates of scanner transmission logs via WebSocket connections.
  • Statistics Dashboard: Comprehensive overview with key metrics and performance indicators.
  • Advanced Filtering: Interactive filtering and search capabilities for log data analysis.
  • Responsive Design: Optimized for various screen sizes and devices.
  • Data Visualization: Charts and graphs for better understanding of transmission patterns.
  • User Management: Role-based access control for different user types.
  • Export Functionality: Ability to export filtered log data for further analysis.
