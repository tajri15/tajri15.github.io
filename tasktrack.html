TaskTrack Pro - Task Management App

Project Summary

TaskTrack Pro is a full-stack task management application built from scratch. It features a modern and responsive frontend using React and Tailwind CSS, powered by a robust backend with Node.js and Express. The application includes a complete, secure local authentication system (registration, login, logout, and password management) using Passport.js and session management with a PostgreSQL database.

Technologies Used

  • Frontend: React 18, TypeScript, Vite, Tailwind CSS, Shadcn/UI, TanStack Query
  • Backend: Node.js, Express.js, TypeScript, TSX
  • Database: PostgreSQL, Drizzle ORM
  • Authentication: Passport.js (Local Strategy), Express-session, Bcrypt.js

Challenge & Solution

Challenge

Building a secure, stateful, full-stack application from the ground up, including a custom authentication system without relying on third-party providers, and ensuring a seamless, real-time user experience.

Solution

Implemented a classic session-based authentication flow with password hashing. Utilized TanStack Query on the frontend for efficient server state management, providing optimistic updates and a smooth user experience. The entire application is strongly typed with TypeScript to ensure code quality and maintainability across the stack.

Main Features

  • User Authentication: Complete, secure registration and login system with password hashing.
  • Profile Management: Users can view their profile and update their name and password.
  • Full CRUD for Tasks: Create, read, update, and delete tasks with properties like priority and due dates.
  • Interactive UI: Real-time filtering, sorting, and searching of tasks directly on the frontend.
  • Dark Mode: A fully implemented dark mode toggle for user comfort.
  • Dashboard Widgets: Includes cards for statistics, recent activities, and upcoming schedules.
