Inventory Management System for PT Telkom

Project Summary

This application is a web-based inventory management system developed for PT Telkom Indonesia. Its goal is to digitize the asset tracking process, from procurement (incoming items) to distribution (outgoing items), thereby reducing manual errors and improving stock management efficiency.

Technologies Used

  • Backend: Python, Flask Framework
  • Database: SQLite (with SQLAlchemy ORM)
  • Frontend: HTML, Jinja2 Template Engine, CSS, JavaScript, Bootstrap 5
  • Key Features: Session-based authentication system, pagination, filtering.

Challenge & Solution

Challenge

Creating a system that is easy to use for non-technical staff with varying levels of digital literacy.

Solution

Designing an intuitive interface with visual guides, clear input validation, and brief training for users. Implemented contextual help features and tooltips to guide users.

Main Features

  • Statistics Dashboard: Displays a summary of key data such as total items, number of categories, and total incoming/outgoing items in the last 30 days.
  • Item Management (CRUD): Full functionality to add, read, update, and delete inventory item data.
  • Dynamic Search and Filter: Users can easily search for items by name or code, as well as filter by category.
  • Category Management: Allows admins to group items into relevant categories for more structured management.
  • Incoming & Outgoing Goods Recording: A dedicated module for recording every transaction of goods coming in from suppliers and goods going out for internal use.
  • Activity Log: Every important action (Create, Update, Delete) performed by a user is recorded in a log for audit and tracking purposes.
